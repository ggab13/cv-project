## Installation 

1. - Clone the repo
 ```sh
git clone https://github.com/ggab13/cv-project.git
   ```

2. - Navigate to the project directory
```sh
 cd cv-project  
 ```
3. - Install dependencies
 ```sh
npm install
```
4. - Navigate to dist folder
 ```sh
 cd cv-project  
 ```
5. - Start the app
  ```sh
npm start
 ```
